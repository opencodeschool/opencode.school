OpenCode is an open source AI coding agent that runs on your machine. It can read your codebase, write code, run commands, and help you build software faster. Think of it as a pair programmer that lives in your editor.

The easiest way to get started is with the **OpenCode Desktop app**. It gives you a graphical interface for chatting with the agent, reviewing its work, and managing your projects. No terminal required.

## Installation

Download the Desktop app for your platform from [opencode.ai](https://opencode.ai). It's available for macOS, Windows, and Linux.

Once installed, open the app and you'll be greeted with a chat interface. You can start typing immediately — no configuration needed. OpenCode includes free models out of the box, so you can start building right away.

## For terminal users

If you prefer the command line, OpenCode also has a terminal app (TUI). Install it with:

```bash
curl -fsSL https://opencode.ai/install | bash
```

But for this course, we'll focus on the Desktop app. It's friendlier for beginners, and everything you learn applies to both interfaces.
