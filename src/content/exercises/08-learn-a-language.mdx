Language models are natural translators — they understand dozens of languages and can move between them fluidly. In this exercise, you'll pair that translation ability with [ElevenLabs](https://elevenlabs.io) text-to-speech to create a personalized language tutor. Pick a language, learn useful phrases, and hear them pronounced correctly — all from OpenCode.

You'll translate phrases with the LLM, then use [ElevenLabs v3](https://replicate.com/elevenlabs/v3) (their most expressive TTS model, supporting 70+ languages) on [Replicate](https://replicate.com) to generate audio you can listen to and practice with.

## Prerequisites

This exercise uses the same setup as [Run AI models](/exercises/run-ai-models). You need:

- The **Replicate skill** installed. If you don't have it yet:

```
npx skills add https://github.com/replicate/replicate-skill
```

- A **Replicate API token** set in your environment. See the [Run AI models](/exercises/run-ai-models) exercise for setup instructions.

Restart OpenCode Desktop after installing the skill or adding the token.

## How it works

The exercise combines two capabilities:

- **Translation** — the LLM translates English phrases to your target language, with phonetic guides so you know how to pronounce them and grammar notes so you understand the structure
- **Text-to-speech** — ElevenLabs v3 speaks the translated text in the target language, so you can hear correct pronunciation and intonation

Portuguese is the default, but you can pick any language ElevenLabs supports — Spanish, French, Japanese, Korean, Hindi, Arabic, and [many more](https://replicate.com/elevenlabs/v3).

## What you'll do

- Choose a language and a topic (greetings, travel, food, small talk)
- Get translations with pronunciation guides and grammar notes
- Hear each phrase spoken aloud by ElevenLabs, with audio files saved locally
- Build a vocabulary file that grows as you learn more phrases
