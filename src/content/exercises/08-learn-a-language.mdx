This exercise covers a lot of ground. You'll use an LLM to translate phrases into a new language, [ElevenLabs](https://elevenlabs.io) to speak them aloud, and [Hono](https://hono.dev) to build a local phrasebook app where you can listen to your phrases and quiz yourself.

[ElevenLabs v3](https://replicate.com/elevenlabs/v3) supports 70+ languages, including Spanish, Portuguese, French, Italian, German, Japanese, Korean, Mandarin, Hindi, Arabic, Turkish, Polish, and many more. The phrasebook grows as you learn, and you can optionally deploy it to [Cloudflare Workers](https://workers.cloudflare.com) when you're ready.

## Prerequisites

This exercise requires the Replicate skill and API token. If you haven't done the [Run AI models](/exercises/run-ai-models) exercise yet, complete that first. It walks through the full setup.

## What you'll build

**Pick a language and learn phrases.** Choose a language and a topic: greetings, ordering food, asking for directions, small talk, whatever interests you. The agent translates English phrases with phonetic pronunciation guides (so you can read the sounds aloud) and grammar notes (so you understand the structure, not just parrot it back).

**Hear them spoken.** For each phrase, the agent uses [ElevenLabs v3](https://replicate.com/elevenlabs/v3) on [Replicate](https://replicate.com) to generate audio in the target language. The MP3 files are saved locally so you can replay them anytime.

**Build a phrasebook app.** The agent scaffolds a [Hono](https://hono.dev) web app that serves your phrasebook as an interactive page. It displays all your phrases in a table with play buttons for each audio clip. Run it locally with `npx wrangler dev` and open it in your browser.

**Flashcard mode.** The app includes a mode that hides translations so you can test yourself. Listen to the audio, try to remember the meaning, then reveal the answer.

**Keep going.** Add more phrases, try different topics, switch languages. Each round the agent translates new phrases, generates audio, and updates the app. Refresh the page to see your phrasebook grow.

## Going further

- Deploy the app to [Cloudflare Workers](https://workers.cloudflare.com) for a permanent, shareable phrasebook
- Try the same phrases across multiple languages to compare how they sound
- Combine with the [Transcribe speech](/exercises/transcribe-speech) exercise: record yourself attempting a phrase and compare your pronunciation to the original
