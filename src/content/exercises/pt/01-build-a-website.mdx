## As ferramentas

Vais usar três ferramentas para construir e publicar o teu site:

**[Hono](https://hono.dev)** é uma framework web pequena e rápida que corre no Cloudflare Workers. Trata do encaminhamento (routing), serve as tuas páginas e suporta JSX para que possas escrever o teu HTML num estilo baseado em componentes. Pensa nela como uma alternativa leve ao Express ou Next.js, feita especificamente para runtimes na edge.

**[Tailwind CSS](https://tailwindcss.com)** é uma framework CSS baseada em utilitários. Em vez de escreveres CSS personalizado, estilizas elementos com classes como `text-lg`, `bg-blue-500` e `rounded-xl`. Neste exercício, vais carregar o Tailwind a partir de um CDN com uma única tag `<script>` — sem passo de compilação, sem ficheiros de configuração, sem npm install. É uma escolha prática para projetos pequenos e protótipos onde queres avançar rapidamente.

**[Cloudflare Workers](https://developers.cloudflare.com/workers/)** é uma plataforma serverless que executa o teu código na edge — perto dos teus visitantes, em todo o mundo. A publicação é feita com um único comando (`npx wrangler deploy`), e o plano gratuito é suficientemente generoso para projetos pessoais.

## Pré-requisitos

Precisas de ter o [Node.js](https://nodejs.org) instalado para correr o servidor de desenvolvimento e publicar o site. Se não o tens:

- **macOS**: `brew install node` (requer o [Homebrew](https://brew.sh))
- **Windows/Linux**: descarrega o instalador em [nodejs.org](https://nodejs.org)

Também precisas de uma conta gratuita no [Cloudflare](https://dash.cloudflare.com/sign-up) — podes registar-te rapidamente com uma conta Google. Se não tens uma, o agente ajuda-te a configurá-la.

## O que vais construir

Uma página pessoal — uma página única com o teu nome, uma biografia curta, os teus interesses e ligações para os teus perfis ou projetos. Vais dar-lhe um aspeto cuidado com Tailwind e publicá-la online.

Quando o básico estiver a funcionar, podes ir mais longe: adicionar mais páginas, criar algo dinâmico com [Workers KV](https://developers.cloudflare.com/kv/) (como um contador de visitas), ou apontar um domínio personalizado para o site.
