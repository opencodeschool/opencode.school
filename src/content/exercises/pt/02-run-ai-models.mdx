Neste exercício, vais usar modelos de IA para gerar imagens e vídeo, diretamente a partir do OpenCode. Vais trabalhar com o [Replicate](https://replicate.com), uma plataforma que aloja milhares de modelos de IA open-source e te permite executá-los através de uma API.

Vais gerar imagens com o [Nano Banana 2](https://replicate.com/google/nano-banana-2) (o modelo de geração de imagens da Google construído sobre o Gemini Flash), e depois transformar uma dessas imagens num vídeo com o [Veo 3.1 Fast](https://replicate.com/google/veo-3.1-fast) (o modelo de geração de vídeo da Google).

## Pré-requisitos

Este exercício requer a **skill Replicate** para o OpenCode. Se completaste a [lição de Skills](/lessons/skills), já a tens instalada. Se não, instala-a agora:

```
npx skills add https://github.com/replicate/replicate-skill
```

Também precisas de uma **conta Replicate** e token de API:

1. Regista-te em [replicate.com](https://replicate.com) (requer uma conta GitHub)
2. Gera um token de API em [replicate.com/account/api-tokens](https://replicate.com/account/api-tokens)
3. Adiciona o token ao teu perfil de shell (`~/.zshrc` ou `~/.bashrc`):

```
export REPLICATE_API_TOKEN=r8_your_token_here
```

Reinicia o OpenCode depois de adicionares o token para que a nova variável de ambiente seja reconhecida.

## O que vais fazer

**Gerar imagens** — Descreve o que queres e o Nano Banana 2 cria-o. Lida com cenas fotorrealistas, ilustrações, maquetas de produtos, entre outros. Também consegue editar imagens existentes: passa uma foto juntamente com um prompt de texto para mudar fundos, trocar estilos, ou combinar múltiplas imagens.

**Gerar vídeo** — Pega numa das tuas imagens geradas e dá-lhe vida com o Veo 3.1 Fast. Descreve o movimento ou cena que pretendes, e o modelo produz um clipe de vídeo curto.

A skill Replicate dá ao OpenCode tudo o que precisa para descobrir modelos, verificar os seus esquemas de entrada, criar previsões e obter resultados, tudo através da API. Sem ferramentas CLI ou SDKs necessários.
