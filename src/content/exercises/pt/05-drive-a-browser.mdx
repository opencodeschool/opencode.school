A maioria das ferramentas de automação de navegadores requer um navegador headless, sessões separadas e muito código. Mas a partir do Chrome 144, o teu agente de IA pode controlar o teu navegador real — o que usas todos os dias, com todos os teus separadores abertos e todas as tuas contas com sessão iniciada.

Isto funciona através do servidor [Chrome DevTools MCP](https://www.npmjs.com/package/chrome-devtools-mcp), que liga o OpenCode ao protocolo de depuração remota do Chrome. O Chrome mostra uma janela de permissão de cada vez e exibe um banner enquanto o agente está ligado, para que nada aconteça em silêncio.

## Configurar o Chrome DevTools MCP

São necessários dois passos:

**Ativar a depuração remota no Chrome** — navega até este URL no Chrome e ativa a funcionalidade:

```
chrome://inspect/#remote-debugging
```

**Adicionar o servidor MCP à tua configuração do OpenCode** — adiciona isto à tua configuração global (`~/.config/opencode/opencode.json`):

```json
{
  "mcp": {
    "chrome-devtools": {
      "type": "local",
      "command": ["npx", "-y", "chrome-devtools-mcp@latest", "--autoConnect"]
    }
  }
}
```

Reinicia o OpenCode depois de fazeres esta alteração.

## Começar no Modo Plano

Antes de deixares o agente interagir com o teu navegador, muda para o **Modo Plano** (Plan Mode). Neste modo, o agente descreve o que pretende fazer e espera pela tua aprovação antes de agir. Esta é uma boa prática de segurança — o teu navegador tem acesso às tuas contas, aos teus dados e às tuas sessões. Queres rever as ações antes de serem executadas.

## A funcionalidade chave: a tua sessão existente

A razão pela qual isto é poderoso é que o teu agente herda a tua sessão existente do Chrome. Já tens sessão iniciada em todo o lado. Isso abre sites que nunca foram pensados para ser automatizados — sites sem API, sem CLI, e com uma interface autenticada complexa:

- Verificar as tuas transações bancárias e categorizar gastos
- Resumir um artigo com acesso restrito ao qual tens subscrição
- Extrair datas de entrega de trabalhos de um portal escolar
- Executar uma auditoria Lighthouse de acessibilidade em qualquer página
- Preencher um formulário longo com o teu agente a guiar-te passo a passo
- Comparar detalhes de produtos em vários separadores

## O que vais fazer

Vais começar de forma simples — tirar uma captura de ecrã do teu separador atual para verificar que a ligação funciona. Depois experimenta uma auditoria Lighthouse. A partir daí, o exercício é aberto: escolhe algo que realmente queiras automatizar e fá-lo.
