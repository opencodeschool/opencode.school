Todas as grandes empresas de tecnologia executam modelos de conversão de voz para texto na cloud, mas podes fazer o mesmo localmente no teu próprio computador — sem chaves de API, sem limites de envio, sem preocupações de privacidade. Neste exercício, vais usar o [Whisper](https://github.com/openai/whisper), o modelo open-source de reconhecimento de voz da OpenAI, para transcrever áudio para texto.

Vais usar o [whisper-cpp](https://github.com/ggerganov/whisper.cpp), uma versão rápida em C++ do modelo original em Python. Corre inteiramente no teu CPU (ou GPU, se disponível), e a instalação via Homebrew é um único comando.

## Instalar as ferramentas

Precisas de duas ferramentas:

```
brew install whisper-cpp ffmpeg
```

O [whisper-cpp](https://github.com/ggerganov/whisper.cpp) executa o modelo de transcrição. O [FFMPEG](https://ffmpeg.org) converte ficheiros de áudio para o formato WAV a 16kHz que o Whisper requer. Se fizeste o exercício [Editar vídeos](/exercises/edit-videos), já tens o FFMPEG instalado.

No Linux, compila o whisper-cpp a partir do código-fonte ou usa o gestor de pacotes do teu sistema. Consulta o [repositório do whisper.cpp](https://github.com/ggerganov/whisper.cpp) para instruções.

## Descarregar um modelo

O Whisper vem em vários tamanhos — desde tiny (75MB) até large (3GB). Vais descarregar uma versão quantizada do modelo large que oferece um bom equilíbrio entre precisão e tamanho do ficheiro (~1GB):

```
curl -o ggml-large-v3-q5_0.bin -L \
  'https://huggingface.co/ggerganov/whisper.cpp/resolve/main/ggml-large-v3-q5_0.bin?download=true'
```

## Obter áudio

Precisas de áudio com voz para transcrever. Escolhe a opção que te parecer mais fácil:

- **Grava-te a ti próprio** — usa o QuickTime Player no macOS (Ficheiro > Nova Gravação de Áudio), ou qualquer aplicação de gravação
- **Reutiliza o áudio do exercício Editar vídeos** — se já tens uma faixa de áudio extraída, usa-a
- **Descarrega algo** — um excerto de podcast, um discurso, uma palestra, qualquer coisa com palavras faladas claramente

## O que vais fazer

- Converter o teu áudio para o formato WAV a 16kHz que o Whisper espera
- Executar a transcrição e obter um resultado de texto com marcas temporais
- Exportar a transcrição como legendas SRT ou VTT — os formatos padrão usados para legendas de vídeo

O Whisper também consegue **traduzir** voz de outras línguas para inglês. Se tens áudio noutro idioma, podes experimentar isso também.
