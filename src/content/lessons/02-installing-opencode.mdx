In this course we'll be using **OpenCode Desktop** — a native app with a [GUI](/glossary#gui) (graphical user interface) you interact with using your mouse and keyboard. If you've used apps like ChatGPT or Claude, this chat-based interface should feel pretty familiar to you.

OpenCode also has a [TUI](/glossary#tui) for terminal-based workflows, a [CLI](/glossary#cli) for one-off commands, and IDE extensions for VS Code, Cursor, Zed, and Windsurf. They all share the same engine and [config](/glossary#config), so what you learn here applies everywhere. But for this course, **Desktop is all you need**.

## Launch the app

Open OpenCode Desktop from your applications menu or dock. You'll be prompted to open a project — click **Open Project** and select any folder on your computer. You should then see a chat interface with a text input at the bottom.

If you run into issues, check the [OpenCode troubleshooting guide](https://opencode.ai/docs/troubleshooting/) or ask for help on the [OpenCode Discord](https://opencode.ai/discord).

## Your first prompt

Once the app is open, try this prompt to confirm everything is working:

> Hello, are you there?

Press **Return** (or **Enter** on Windows) to send. If you get a response, you're connected to a language model and ready to go.
