Before we install anything, let's make sure you have what you need:

- A new-ish computer running macOS, Windows, or Linux
- A reliable internet connection
- A text editor for viewing and editing configuration files

## A new-ish computer

OpenCode Desktop runs on **macOS**, **Windows**, and **Linux**. Any modern computer from the last 5-6 years should work fine. OpenCode lets your run AI models that are hosted in the cloud, so you don't need a powerful machine — just something that can run a web browser and connect to the internet. You can also configure OpenCode to use models running locally on your own machine using tools like Ollama and LM Studio, but that's optional and not required for completing the OpenCode School curriculum.

## Text editor

You'll need a text editor for viewing and editing OpenCode's configuration files. A text editor is a program for writing and editing plain text files. It doesn't have to be anything fancy — just something that can open `.txt` or `.md` files.

Here are few recommendations, all of which are freely available:

- [VS Code](https://marketplace.visualstudio.com) - the world's most popular code editor, with a huge ecosystem of extensions and features. Great for both beginners and experienced developers.
- [Cursor](https://cursor.com) - A fork of VS Code with built-in AI features and a more streamlined interface. 
- [Zed](https://zed.dev) - A new editor built for speed and simplicity, with a focus on real-time collaboration. 
- [Windsurf](https://windsurf.com/editor) - A lightweight editor with built-in AI code generation and autocompletion.

If you don't want to install anything extra, you can use the text editor that comes with your computer:

- **macOS**: TextEdit (in your Applications folder). Go to TextEdit > Settings and select "Plain Text" as the default format — otherwise it will try to save files as rich text.
- **Windows**: Notepad (search for it in the Start menu).
- **Linux**: Varies by distribution — gedit (GNOME), Kate (KDE), mousepad (XFCE), or `nano` in the terminal.

OpenCode itself will handle most file creation and editing. The text editor is mainly for when you want to manually inspect or tweak [config](/glossary#config) files.

## That's it!

No special software to install yet, no accounts to create. If you have a computer with an internet connection and a text editor, you're ready for the next lesson.
